# Beetfolio

近未来トーンの個人資産管理ダッシュボード。
暗号資産・株式/投信・銀行/現金をまとめて俯瞰し、将来は取引所/証券/銀行APIと自動連携する。

## 現状（第1段階：デザインプロトタイプ）

ブラウザで即開ける静的プロトタイプ。データはすべて `app.js` 内のダミー。

| ファイル | 役割 |
|---|---|
| `index.html` | 画面構造（サイドバー / KPI / チャート / 口座 / 配分 / 保有 / 取引） |
| `styles.css` | デザイントークンとレイアウト（ダーク + バイオレット×マゼンタ×シアン） |
| `app.js`     | ダミーデータ定義 と SVG チャート/ドーナツの描画。**ここをAPIに差し替える** |
| `server.js`  | 依存なしの静的サーバ（プレビュー用） |

### 起動

```bash
node server.js
# → http://localhost:4173
```

（`index.html` をダブルクリックで直接開いても動作。フォントのCDN取得のみネット接続が必要）

## ロードマップ

- **第2段階 — フレームワーク移植**: Next.js + TypeScript + Tailwind に再構成。
  コンポーネント分割（KpiCard / NetWorthChart / AccountList など）、チャートは Recharts か現状のSVGを継承。
- **第3段階 — データ層**: DB（Postgres / SQLite）+ Prisma。口座・残高・取引・評価額のスキーマ定義。
- **第4段階 — API連携**:
  - 暗号資産: 取引所API（bitFlyer / Binance 等）の残高・約定取得
  - 株式/投信: 証券口座 or 株価API（評価額の自動更新）
  - 銀行/現金: 残高の取込（手動入力 or 連携）
  - 秘密情報はサーバ側で暗号化保管。読み取り専用APIキーを基本とする。
- **第5段階 — 機能拡張**: 損益分析、リバランス提案、目標資産トラッキング、通知。
