# Beetfolio

近未来トーンの個人資産管理ダッシュボード。
暗号資産・株式/投信・銀行/現金をまとめて俯瞰し、将来は取引所/証券/銀行APIと自動連携する。

## 現状（第1.5段階：機能するSPA）

依存ライブラリなしの SPA。データは **localStorage に永続化**（リロードしても残る）。
すべてのボタンが動作し、口座・保有・取引の追加/編集/削除ができる。

### 主な機能
- **8ページ** — ダッシュボード / 総資産 / 暗号資産 / 株式・投信 / 銀行・現金 / 分析 / 入出金 / 設定（ハッシュルーティング）
- **CRUD** — 口座・保有銘柄・取引の追加/編集/削除
- **取引の自動反映** — 売買/入出金を記録すると、保有数量と平均取得単価が自動更新
- **算出指標** — 総資産・評価損益・現金比率・資産配分はすべて保有データから動的計算
- **価格更新（同期）** — ⟳ ボタンで価格をシミュレート更新
- **検索 / 通知 / プロフィール** メニュー、操作トースト通知
- **データ管理** — JSON エクスポート / インポート / 初期化

### ファイル構成
| ファイル | 役割 |
|---|---|
| `index.html`    | アプリシェル（サイドバー / トップバー / `#view`） |
| `styles.css`    | デザイントークン・レイアウト・全コンポーネント |
| `store.js`      | データ層。localStorage 永続化・CRUD・セレクタ。**将来ここをAPI/DBに差し替える** |
| `charts.js`     | SVGチャート（折れ線・ドーナツ・スパークライン・バー） |
| `components.js` | UI部品（モーダル・フォーム・トースト・確認・ポップオーバー・整形） |
| `app.js`        | ルーター・各ページの描画・イベント処理 |
| `server.js`     | 依存なしの静的サーバ（プレビュー用） |

### 起動
```bash
node server.js   # → http://localhost:4173
```

## ロードマップ
- **第2段階 — フレームワーク移植**: Next.js + TypeScript + Tailwind に再構成。`store.js` の構造をそのまま型・APIクライアントに昇格。
- **第3段階 — データ層**: DB（Postgres / SQLite）+ Prisma。accounts / holdings / transactions スキーマは現状の構造を踏襲。
- **第4段階 — API連携**: 取引所（bitFlyer/Binance）・証券/株価・銀行残高。読み取り専用キー＋サーバ側暗号化。
- **第5段階 — 機能拡張**: 損益分析の高度化、リバランス提案、目標トラッキング、通知。
